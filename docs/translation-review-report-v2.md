## 검토 개요
- **검토 대상**: 전체 66권 성경 번역본
- **대상 연령**: 8-10세 (초등학교 저학년)
- **검토 방법**: 영어 원문(WEB)과 한글 번역 직접 비교

  5. 일관성

### 4.2 Yahweh 번역 일관성 문제

현재 상태:
- 대부분: "하나님"으로 번역 ✅
- 일부 구절: "여호와"가 남아있음 (검색 결과 0건으로 대부분 정리됨)
- 일부 구절: "주 하나님", "만군의 하나님" 등 다양한 표현 사용

**권장**: "하나님"으로 통일 유지

### 4.3 로마서 등 서신서 난이도

신학적 개념이 많아 어린이에게 여전히 어려움:

**권장**: 서신서는 어린이용 요약본 별도 제작 고려

## 5. 수정 우선순위

### 🔴 긴급 (즉시 수정)

1. **잠언 전체**: "하나님 하나님" → "하나님" 일괄 수정 (50+ 건)
2. **전체 파일**: "하나님는" → "하나님은" 일괄 수정 (30+ 건)
3. **전체 파일**: "하나님를" → "하나님을" 일괄 수정 (25+ 건)

### 🟡 중요 (단기 수정)

1. "대적" → "나를 미워하는 사람들" 일관성 있게 순화
2. 시편의 "Selah" 처리 확인 (현재 대부분 생략됨 - 적절함)
3. 종교 용어에 괄호 설명 추가 검토

### 🟢 권장 (장기 개선)

1. 서신서 난이도 재검토
2. 15단어 이상 긴 문장 분리
3. 전체 존댓말 일관성 확인 ("~해요" 체)

## 7. 수정 스크립트 예시

```python
# 긴급 수정 사항 일괄 처리
import json
import glob

files = glob.glob('korean/*.json')

for file in files:
    with open(file, 'r', encoding='utf-8') as f:
        content = f.read()

    # 수정 사항
    content = content.replace('하나님 하나님', '하나님')
    content = content.replace('하나님는', '하나님은')
    content = content.replace('하나님를', '하나님을')

    with open(file, 'w', encoding='utf-8') as f:
        f.write(content)
```
