# 어린이 성경 번역 품질 검토 보고서

## 검토 개요
- **검토 대상**: 전체 66권 성경 번역본
- **대상 연령**: 8-10세 (초등학교 저학년)
- **검토 기준**:
  1. 어린이가 이해하기 어려운 단어
  2. 어색하거나 부자연스러운 문장
  3. 번역 일관성

## 2. 책별 주요 문제점

### 2.1 구약 - 모세오경

#### 창세기
- **3:24** "그룹 천사" → "특별한 천사들" + 설명 필요
- **6:14** "역청" → "검은 끈끈한 물질"
- **19:31-36** 롯과 딸들 이야기 - 8-10세에게 부적절, 표현 완곡하게 수정 필요

#### 출애굽기
- **1:15** "산파" → "아기를 받아주는 사람들"
- **7:1** "대언자" → "대신 말해주는 사람"
- **12:22** 유월절 설명 문장이 너무 길고 복잡 → 문장 분리 필요

#### 레위기 ⚠️ 주의 필요
- 제사 용어(속죄제, 속건제, 번제, 소제, 화목제 등)가 매우 많음 → 각 용어마다 간단한 설명 추가 필요
- 13장 피부병 진단 - 너무 의학적, 요약 필요

#### 민수기
- 1장 숫자와 명단 반복 - 어린이에게 요약본 필요

#### 신명기
- **1:15** "천부장, 백부장, 오십부장, 십부장" → 괄호로 설명 추가

### 2.2 구약 - 역사서

#### 공통 문제
- "성읍" → "마을" 또는 "도시"로 통일
- "족보" → "가족 기록" 또는 "조상 목록"

#### 에스더
- **4:14** "당신이 이때를 위해 왕비가 된 것인지 누가 알겠어요?" → "어쩌면 당신이 이런 때를 위해 왕비가 되었을지도 몰라요"

### 2.3 구약 - 시가서

#### 욥기 ⚠️ 주의 필요
- **16:13** "내 콩팥을 쪼개시고... 내 쓸개를 땅에 쏟으셨어요" - 너무 폭력적, 어린이에게 부적절
- **20:7** "자기 똥처럼" - 직접적인 표현, 품위 문제

#### 시편
- "원수", "대적" 매우 빈번 → "나를 미워하는 사람들"로 통일

#### 잠언 ⚠️ 주의 필요
- **5장, 7장** - 성적 유혹에 관한 내용, 8-10세에게 매우 부적절
- **6:26** "창녀" - 직접적, 부적절

#### 전도서
- 전체 톤이 염세적, 허무주의적 - 어린이에게 적절한지 재검토 필요

#### 아가 ⚠️ 주의 필요
- **전체** - 성인의 낭만적 사랑에 대한 시로 8-10세에게 부적절, 생략 권장

### 2.4 구약 - 대선지서

#### 이사야
- **1:21** "창녀가 되었는가!" → "하나님께 등을 돌리게 되었나요!"
- **3:16** "정을 통하는 눈" → "자랑스러운 눈"

#### 예레미야
- **2:24** 발정기 관련 표현 - 매우 부적절, 삭제 또는 대폭 수정 필요

### 2.5 구약 - 소선지서

#### 미가 ⚠️ 주의 필요
- **3:2-3** "백성의 가죽을 벗기고 뼈에서 살을 뜯어내는구나..." - 매우 폭력적, 완화 필요

#### 말라기 ⚠️ 주의 필요
- **2:3** "너희 얼굴에 똥을 뿌리겠다" - 매우 거칠고 부적절, "너희를 부끄럽게 만들겠다"로 수정

#### 나훔
- **3:5** "나는 네 치마를 네 얼굴까지 들어 올릴 거야" → "모두가 보는 앞에서 부끄럽게 만들 거야"

### 2.6 신약 - 복음서

#### 마태복음
- **5:27-28** "간음", "음욕" - 8-10세에게 설명하기 어려운 개념
- 종교 직분/제도 용어(대제사장, 율법학자, 바리새인 등) - 각주 또는 괄호 설명 필요

#### 누가복음
- **2:7** "구유" → "동물이 먹이를 먹는 통" 설명 추가

### 2.7 신약 - 서신서

#### 전체 서신서 공통
- 추상적 신학 개념이 매우 많음 (의롭게 됨, 거룩함, 구속, 화해 등)
- 직역된 문장 구조가 길고 복잡함

#### 히브리서 ⚠️ 주의 필요
- **5-10장** 멜기세덱, 대제사장, 제사, 언약 등 신학적 내용이 8-10세에게 너무 어려움
- **7장 전체** 멜기세덱 설명이 어린이에게 이해 불가 수준

#### 베드로후서
- **2:22** "개가 자기 토한 것을 다시 먹는다" - 충격적일 수 있음

## 3. 일관성 문제

### 3.1 하나님 호칭 혼용
| 현재 상태 | 권장 |
|-----------|------|
| "하나님", "여호와", "주님", "주 하나님" 혼용 | 어린이 성경은 "하나님"으로 통일 권장 |

### 3.2 동일 개념의 다른 표현
| 개념 | 현재 표현들 | 권장 |
|------|------------|------|
| 복음 | 기쁜 소식, 좋은 소식 | 하나로 통일 |
| 가져가다 | 가져가라, 취하라, 데려가라 | "가져가세요"로 통일 |
| 신발 | 신발 끈 풀기, 신발 들고 다니기 | 원문에 따라 정확히 |

### 3.3 존댓말/반말 혼용
- "~해야 해요" / "~해라" / "~하여라" 혼용됨
- **권장**: "~해요", "~하세요" 등 존댓말로 통일

## 4. 우선순위별 수정 권장사항

### 🔴 긴급 수정 필요 (어린이에게 부적절)

1. **잠언 5장, 7장** - 성적 유혹 내용 삭제 또는 대폭 수정
2. **아가 전체** - 8-10세용으로 부적절, 생략 권장
3. **미가 3:2-3** - 폭력적 표현 완화
4. **말라기 2:3** - 거친 표현 수정
5. **욥기 16:13** - 폭력적 표현 완화
6. **예레미야 2:24** - 부적절한 표현 삭제/수정
7. **창세기 19:31-36** - 롯과 딸들 이야기 완곡하게 수정

### 🟡 중요 수정 필요 (이해 어려움)

1. **한자어 순화** - 위 목록의 한자어들을 쉬운 우리말로 변경
2. **종교 용어 설명** - 제사 용어, 종교 직분에 괄호로 간단한 설명 추가
3. **하나님 호칭 통일** - "하나님"으로 일관되게 수정
4. **측정 단위 설명** - 규빗, 세겔, 달란트 등에 현대적 비교 추가

### 🟢 개선 권장 (품질 향상)

1. **긴 문장 분리** - 15단어 이상 문장 분리
2. **비유 설명 추가** - 어려운 비유에 간단한 설명 추가
3. **존댓말 통일** - "~해요" 체로 일관되게 수정
4. **고대 문화 설명** - 타작마당, 구유 등에 간단한 설명 추가

## 5. 전체 평가

### 잘된 점
- 전반적으로 "~해요" 체를 사용하여 어린이 친화적인 톤 유지
- 주요 고유명사(하나님, 예수님, 모세 등)는 일관되게 번역됨
- 요나, 룻기 등 이야기 형식의 책은 비교적 잘 번역됨

### 개선 필요
- 한자어와 종교 전문 용어가 여전히 많음
- 일부 성인 내용이 완화 없이 번역됨
- 히브리서, 로마서 등 신학적 내용이 어려운 책은 추가 순화 필요
- 폭력적이거나 성적인 표현의 완화 필요
